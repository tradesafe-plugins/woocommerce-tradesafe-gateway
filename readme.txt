This is the official WooCommerce extension to receive payments using the TradeSafe as a payment provider. This plugin is originally based on WooCommerce PayFast Gateway.

== Description ==

The TradeSafe extension for WooCommerce enables you to accept payments via one of South Africaâ€™s most popular payment gateways.

= Why choose TradeSafe? =

TradeSafe gives your customers more flexibility including putting down deposits, ordering ahead of time or paying on a weekly, monthly or annual basis.

== Frequently Asked Questions ==

= Does this require a TradeSafe API account? =

Yes! A TradeSafe api account and API key are required for this gateway to function.

= Does this require an SSL certificate? =

An SSL certificate is recommended for additional safety and security for your customers.

= Where can I find documentation? =

For help setting up and configuring, please refer to our [user guide](https://www.tradesafe.co.za/page/API)
